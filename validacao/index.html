O que é Jquery

jQuery é uma biblioteca de JavaScript que simplifica a manipulação do DOM (Modelo de Objeto de Documento) e o desenvolvimento de interações dinâmicas em páginas web.Ela oferece uma sintaxe fácil de usar, tornando mais simples a seleção de elementos HTML, manipulação de eventos, animações e comunicação assíncrona.

Exemplo jQuery


--$(document).ready(function(){...});: Aguarda até que o DOM esteja totalmente carregado antes de executar o código interno.

--$("#meuBotao").click(function(){...});: Atribui uma função ao evento de clique do botão com o ID "meuBotao".

--$("#minhaCaixaDeTexto").val("Texto alterado com jQuery!");: Altera o valor da caixa de texto com o ID "minhaCaixaDeTexto".

--Ao clicar no botão, o conteúdo da caixa de texto será alterado de acordo. Este é apenas um exemplo básico, e o jQuery oferece muitos recursos para manipulação de elementos, eventos, animações e comunicação assíncrona, como AJAX.


Exemplo de Regex:


Regex, é uma sequência de caracteres que define um padrão de busca. Esses padrões são utilizados para encontrar correspondências em strings ou para validar se uma string atende a um determinado critério.O regex e normalmanete usado na programação para operações de busca, substituição e validação de texto.


Exemplo de uso em JavaScript:

javascript

Copy code

// Verifica se a string contém um número de telefone no formato (XXX) XXX-XXXX

const regex = /\(\d{3}\) \d{3}-\d{4}/;


const texto = "Entre em contato pelo telefone (123) 456-7890.";

if (regex.test(texto)) {

console.log("Número de telefone válido.");

} else {

console.log("Número de telefone inválido.");

}


Neste exemplo, a expressão regular \(\d{3}\) \d{3}-\d{4} representa o formato de um número de telefone.

A função test() verifica se a string atende a esse padrão.
